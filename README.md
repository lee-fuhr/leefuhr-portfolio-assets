# leefuhr-portfolio-assets

Image assets for the leefuhr.com 4.0 portfolio case studies. Served via `raw.githubusercontent.com` for Notion drafts and Webflow embeds.

## Layout

```
clients/<slug>/<filename>.png
```

## URL pattern

```
https://raw.githubusercontent.com/lee-fuhr/leefuhr-portfolio-assets/main/clients/<slug>/<filename>.png
```
